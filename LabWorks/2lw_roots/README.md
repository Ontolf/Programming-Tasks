Lab18_roots  
  
In this program, a and b are two borders and c is the value where f(x) = c, where x between a and b.  
  
  Example:    
  a = 2 (value of left border)  
  b = 5 (value of right border)  
  c = 3.42424 (where f(x) = c)  
  br = 0.1 (distence between borders)
