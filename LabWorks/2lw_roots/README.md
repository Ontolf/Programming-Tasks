Lab18_roots  
  
In this program, a and b are two borders and c is the value where f(x) = c, where x between a and b.  
  
  Example:    
  a = -6 (value of the left border)  
  b = 9 (value of the right border)  
  c = 0.8 (where f(x) = c)  
  br = 0.00001 (distence between borders)
